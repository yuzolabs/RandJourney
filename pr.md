feat: RandJourney - ランダムダーツ旅アプリの初期実装。

## 概要

RandJourney は、地図上でランダムな場所を選んで「ダーツ旅」を楽しむための Web アプリケーションです。国土地理院のタイル地図を使用し、指定した範囲内からランダムな地点を選出し、逆ジオコーディングで住所を表示します。結果は X（Twitter）でシェアできます。

この PR では、アプリケーションの基盤となる全機能を実装しています。

## 修正事項

### プロジェクト基盤

- Vite + React + TypeScript のプロジェクト構成をセットアップ
- Vitest + Testing Library によるテスト環境を構築
- Apple HIG（Human Interface Guidelines）に基づいたデザインシステムを導入

### コア機能

- 地図表示 - 国土地理院（GSI）タイル地図の統合
- ランダム地点生成 - WGS84測地系に基づく一様分布によるランダム地点生成
- 逆ジオコーディング - GSI 逆ジオコーディングサービスとの連携（リトライ機能付き）
- 市区町村検索 - 日本の市区町村データによる住所解決

### UI/UX機能

- ダーツ投げフロー - 中心点の設定 → 範囲選択 → ダーツ投げ → 結果表示
- 距離コントロールパネル - ガラスモーフィズムデザインのスライダーUI
- 結果ポップアップ - ダーツ投げ後の結果表示と住所情報
- 履歴機能 - localStorage を使用したダーツ履歴の永続化
- ライト/ダークモード - システム設定に応じたテーマ切り替え
- レスポンシブレイアウト - モバイル・デスクトップ対応

### 共有機能

- X（Twitter）共有 - 結果を X でシェアする機能
- URL 共有 - `?ll=`パラメータによる位置情報の URL 共有

### CI/CD

- GitHub Pages への自動デプロイワークフローを追加

### スキル・ドキュメント

- Apple HIG Designer スキルの追加（HIG 準拠コンポーネント設計用）
- frontend-design スキルの追加

## 備考
